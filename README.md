# Python Console Calculator

Простой консольный калькулятор на Python, поддерживающий базовые арифметические операции и работу в цикле до выхода пользователя.

## Возможности

✅ Поддерживаемые операции:
- `+` — сложение  
- `-` — вычитание  
- `*` — умножение  
- `/` — деление  
- `**` — возведение в степень  
- `%` — остаток от деления  

✅ Обработка ошибок:
- деление на ноль
- неверный ввод чисел
- неизвестная операция

✅ Выход из программы:
- введите `q` на любом шаге ввода
